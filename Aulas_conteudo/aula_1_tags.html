Introdução ao HTML

    Tags básicas:
  • < é usado para abrir uma tag e > ou /> para fechar

  • <h1>...<h6> transforma o texto em título

  • <p> faz um parágrafo com o texto

  • <b> é usado para deixar o texto em negrito

  • <i> é usado para deixar em italico

  • <u> é usado para deixar o texto sublinhado

  • <a> é usado para colocar um link no texto

  • <br> quebra ou adiciona uma linha em branco

  • <img> adiciona uma imagem

  • <ol> cria uma lista ordenada

  • <ol type=""> tipo de numeração da lista

  • <ul> cria uma lista não ordenada
